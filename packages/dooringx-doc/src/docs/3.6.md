## 命令的导入

命令对象导入到插件的initCommandModule里即可

```js
initCommandModule: commandModules,
```

## 命令的开发

命令需要导出一个CommanderItemFactory生成的对象。

```js
import { CommanderItemFactory } from 'dooringx-lib';
const undo = new CommanderItemFactory(
	'redo',
	'Control+Shift+z',
	(store) => {
		store.redo();
	},
	'重做'
);

export default undo;
```
